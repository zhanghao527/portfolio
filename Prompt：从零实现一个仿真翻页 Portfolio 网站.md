# Prompt：从零实现一个仿真翻页 Portfolio 网站

## 项目概述

用 React + TypeScript + Vite 从零构建一个个人作品集（Portfolio）网站，核心特色是**自研的仿真翻页引擎**，模拟微信读书的真实纸张翻页效果。不使用任何第三方翻页库，完全用 Canvas + CSS clip-path + DOM 分层实现。

技术栈：React 18 + TypeScript + Vite + 纯 CSS（无 UI 框架）

---

## 一、翻页引擎核心架构（usePageFlip Hook）

### 1.1 四层渲染结构（从底到顶）

翻页时页面由四个层叠的层组成：

- **Layer 1（z-index: 1）**：下一页内容，完整显示，作为底层
- **Layer 2（z-index: 2）**：当前页内容，用 `clip-path: polygon()` 沿折线裁剪，只显示未被翻起的部分
- **Layer 3（z-index: 3）**：Canvas 层，绘制翻起纸张的背面底色（不透明的 `#eae8e2`）、折痕阴影、投射阴影
- **Layer 4（z-index: 4）**：当前页内容的镜像层，用 CSS `transform: matrix()` 做关于折线的反射变换，`clip-path` 裁剪到纸张背面区域，`opacity: 0.15`，模拟从纸张背面透过去看到的淡淡文字

### 1.2 折线几何计算

翻页的核心几何概念：

- **finger**：当前拖拽点（鼠标/触摸位置，相对于容器）
- **origin**：翻页起始角的原始位置（如右下角 `(w, h)`）
- **折线（fold line）**：finger 和 origin 的**中垂线**
  - 中点 `mid = ((finger.x + origin.x)/2, (finger.y + origin.y)/2)`
  - 方向向量 `dir = (-(origin.y - finger.y), origin.x - finger.x)`（垂直于 finger→origin）
- **折线与页面矩形的交点**：用参数方程求折线与矩形四条边的交点，得到 2 个交点
- **页面分割**：用叉积判断矩形四个角在折线的哪一侧
  - origin 侧 = 被翻起的区域（flipped）
  - 另一侧 = 当前页保留的区域（kept）
  - 遍历矩形边，在折线交叉处插入交点，分别收集两侧的多边形顶点

### 1.3 镜像变换

纸张背面的内容和位置通过**关于折线的反射变换**计算：

- 反射矩阵：给定折线法向量 `(ux, uy)`（finger→origin 方向的单位向量），反射矩阵为：
  ```
  a = 1 - 2*ux²,  b = -2*ux*uy
  c = -2*ux*uy,    d = 1 - 2*uy²
  tx = mid.x - a*mid.x - b*mid.y
  ty = mid.y - c*mid.x - d*mid.y
  ```
- 输出为 CSS `matrix(a, c, b, d, tx, ty)` 用于镜像 DOM 层
- 同样的反射公式用于镜像多边形顶点（计算纸张背面的 clip-path 区域）

### 1.4 Canvas 绘制内容

Canvas 层绘制三个效果：

1. **纸张背面底色**：将 flipped 多边形的镜像区域填充不透明的纸色 `#eae8e2`，加一个从折线向外的渐变阴影
2. **折痕阴影**：沿折线方向画一条 15px 宽的渐变阴影条（`rgba(0,0,0,0.12)` → 透明），模拟纸张弯曲处的阴影
3. **投射阴影**：在镜像区域上用 `ctx.shadowBlur = 20` 画投射阴影

### 1.5 clip-path 输出

- `currentClip`：kept 多边形 → CSS `polygon(x1% y1%, x2% y2%, ...)` 百分比坐标
- `backClip`：mirrored 多边形 → 同上，用于裁剪镜像 DOM 层
- `backTransform`：CSS `matrix()` 字符串，用于镜像 DOM 层的 transform

## 二、交互系统

### 2.1 拖拽翻页

- 检测区域：页面四角和左右边缘各 80px 范围
- 角落检测优先级：右上角 > 右下角 > 右边缘默认为右下角；左侧同理
- 拖拽阈值：鼠标移动 > 6px 才进入拖拽模式
- 拖拽过程：实时更新 `finger` 位置和 `progress`（0~1），每帧重新计算折线、clip-path、Canvas
- 松手判定：
  - `progress > 0.2` → 提交翻页，从当前位置动画到完全翻过去
  - `progress <= 0.2` → 回弹到原始角落
- 松手后动画时长 = `max(150ms, remaining * 400ms)`，距离越短动画越快

### 2.2 点击翻页

- 点击右侧 60% 区域 → 翻下一页
- 点击左侧 40% 区域 → 翻上一页
- 根据点击的 y 位置分三个区域，决定翻页起始角和路径：
  - **上 1/3**：从右上角掀起，`origin=(w,0)` → `target=(-w*0.5, h*0.15)`
  - **中 1/3**：卷轴式平推，`origin=(w,h)` → `target=(-w*0.5, h)`
  - **下 1/3**：从右下角掀起，`origin=(w,h)` → `target=(-w*0.5, h*0.85)`
- 目标点的 y 坐标接近起点 y（偏移约 15%），保持翻页角度小而自然
- 动画时长 1000ms，匀速（`e = t`）
- 防闪烁：先设置状态但 `flipping=false`，下一帧才设 `flipping=true` 并启动动画
- 动画结束：先设 `currentPage=idx`，下一帧再设 `flipping=false`，避免状态切换闪烁

### 2.3 触摸支持

- 同时监听 mouse 和 touch 事件
- touch 事件使用 `{ passive: true }`

## 三、书签导航系统

### 3.1 书签布局

- 书签贴在书本右侧边缘外（已翻过的页面在左侧）
- 固定高度 50px，间距 58px，起始位置 top: 40px
- 竖排文字 `writing-mode: vertical-rl`
- 当前页书签高亮（纸色背景 `#fffef8`），其他页灰色

### 3.2 书签跟随翻页

翻页时，当前页的书签跟着翻起的纸张移动：

- **位置计算**：将书签在原始页面边缘上的两个端点（上端和下端）通过折线反射变换得到镜像位置
- **角度计算**：两个镜像端点的连线方向即为书签的旋转角度
- **外侧偏移**：书签沿镜像边缘的垂直方向向外偏移 40px（书签宽度）
- **边界限制**：镜像位置超出书本范围时固定在边缘
- **延迟启动**：`progress < 0.03` 或折线未到达书签位置时，书签保持原位不动
- **文字方向**：书签整体旋转但文字不额外反转，保持和书签方向一致

### 3.3 书签状态

- `'right'`：未翻过的页面，书签在右侧
- `'left'`：已翻过的页面，书签在左侧
- `'moving'`：正在翻页的页面，书签跟随纸张移动

## 四、页面内容结构

### 4.1 页面列表

1. **封面「关于我」**：居中布局，头像、标题、简介、社交链接，右下角"翻开 →"呼吸动画提示
2. **项目**：项目卡片列表，每个卡片有 emoji 图标 + 名称 + 描述
3. **技术栈**：圆角标签列表
4. **博客**：文章列表，每篇有标签 + 日期 + 标题

### 4.2 页面样式

- 纸张底色 `#fffef8`（微黄白）
- 淡横线纹理（`repeating-linear-gradient`，间距 32px，颜色 `rgba(0,0,0,0.012)`）
- 页码居中显示在底部
- 整体背景 `#c8c3bb`（暖灰色），带微弱的径向渐变光影

## 五、性能优化要点

1. **缓存容器尺寸**：用 `sizeRef` 缓存 `getBoundingClientRect()`，只在 resize 时更新，避免每帧强制布局
2. **动画开始时缓存宽度**：`animateTo` 在启动时读取一次宽度，动画循环中不再调用
3. **Canvas DPR 适配**：`canvas.width = w * dpr`，`ctx.scale(dpr, dpr)`
4. **will-change**：clip-path 层设 `will-change: clip-path`
5. **防闪烁的两阶段状态切换**：翻页开始和结束都分两帧完成状态变更

|------|-----|------|
| 拖拽检测区域 | 80px | 页面四角/边缘的拖拽触发范围 |
| 拖拽阈值 | 6px | 鼠标移动超过此值才进入拖拽模式 |
| 提交阈值 | 0.2 | progress 超过此值松手即提交翻页 |
| 点击翻页时长 | 1000ms | 匀速动画 |
| 翻页目标 x | ±w*0.5 | 目标点在页面外 50%，确保折线完全移出 |
| 翻页目标 y 偏移 | 15% | 角落翻页的 y 偏移量，保持小角度 |
| 纸张背面色 | #eae8e2 | 不透明，完全遮挡下层 |
| 镜像内容透明度 | 0.15 | 纸张背面的文字透视效果 |
| 折痕阴影宽度 | 15px | 折线处的阴影条宽度 |
| 书签尺寸 | 40×50px | 固定尺寸，间距 58px |
| 书签起始位置 | top: 40px | 第一个书签的顶部位置 |
